===다시 로컬 레파지토리에서 작업 

1. 원격 레파지 토리 등록하기 

git remote add origin '원격 레파지토리 주소'

2. 원격 레파지토리 내용을 로컬 레파지토리로 내려 받기 

git pull origin main 

3. 업데이트 된 로컬 레파지토리 내용을 원격 레파지토리로 올리기 
git push origin main
